### solidity masking

```k
rule X |Int 0 => X
  requires #rangeUInt(256, X)

rule chop(A &Int B) => A &Int B
  requires #rangeUInt(256, A)
  andBool #rangeUInt(256, B)

rule chop(A |Int B) => A |Int B
  requires #rangeUInt(256, A)
  andBool #rangeUInt(256, B)

rule (A |Int B) <Int pow256 => true
  requires #rangeUInt(256, A)
  andBool #rangeUInt(256, B)

rule (A &Int B) <Int pow256 => true
  requires #rangeUInt(256, A)
  andBool #rangeUInt(256, B)
```
